# Tetris (pédagogique) — HTML/CSS/JS + Flask

## Lancer en local (sans Python)
- Ouvre `index.html` dans un navigateur (Chrome/Firefox/Edge/Safari).

## Lancer en local (avec Flask)
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python3 app.py
```
Puis ouvre `http://localhost:5000`.

## Déployer (Railway)
- `requirements.txt` + `Procfile` sont inclus.
